# Clarabelle

A Southpaw 1800-mini style board designed by The_Royal of KinglyKeys.

- Keyboard Maintainer: [The-Royal](https://github.com/the-royal)
- [Cannon Keys Groupbuy Product Page](https://cannonkeys.com/products/gb-clarabelle-keyboard)

Make examples for this keyboard (after setting up your build environment):

**Default Layout:**

    make cannonkeys/clarabelle:default
